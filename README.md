# 字義門 · 句型內功（網頁版）

這是一個教小學生國文句型的單頁網頁遊戲範例，包含完整句子、因果、順序、心情、三選一、標點與看圖任務。

如何使用：
- 在本機開啟 `index.html`（瀏覽器即可）。
- 遊戲會把進度儲存在 localStorage，完成題目可獲得內功點數並提升等級徽章。

檔案：
- `index.html`：主頁面
- `css/style.css`：樣式
- `js/app.js`：遊戲邏輯與題庫
- `printable.html`：列印版題庫（可直接列印）
- `docs/lesson_plan.md`：60 分鐘家教流程與評分規準
